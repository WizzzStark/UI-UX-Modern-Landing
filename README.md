# Instructions

## Visit deployed website
https://ui-ux-modern-landing.vercel.app/

## Deploy proyect on localhost
### Clone the proyect
```
git clone https://github.com/WizzzStark/UI-UX-Modern-Landing/
```
### Install dependencies
```
npm install
```
### Run website on localhost:3000
```
npm run dev
```
